## 新增需求
### 需求：面板自由拉伸与宽度记忆
系统必须支持三面板自由拉伸，并持久化各面板宽度。

#### 场景：拖拽调整宽度
- **当** 用户拖拽面板分割条
- **那么** 对应面板宽度立即更新并记录

#### 场景：重新打开应用
- **当** 应用重启
- **那么** 面板宽度恢复到上次值

### 需求：一键收缩与主视图切换
系统必须支持一键收缩/展开，并允许终端或任务为主视图显示。

#### 场景：收缩任务面板
- **当** 用户点击任务面板收缩按钮
- **那么** 任务面板隐藏，终端面板自动扩展为主视图

#### 场景：收缩终端面板
- **当** 用户点击终端面板收缩按钮
- **那么** 终端面板隐藏，任务面板自动扩展为主视图

### 需求：文件面板与任务面板联动
系统必须保证文件面板与任务面板联动收缩与展开。

#### 场景：收缩文件面板
- **当** 用户点击文件面板收缩按钮
- **那么** 文件面板隐藏且任务面板成为左侧主视图

#### 场景：展开文件面板
- **当** 用户点击文件面板展开按钮
- **那么** 文件面板恢复并保持上次宽度
